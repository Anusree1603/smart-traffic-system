# Smart Traffic System

## 📌 Description
This project is an AI and IoT-based traffic management system that detects emergency vehicles and adjusts traffic signals dynamically while monitoring pollution levels.

## 🚀 Features
- Emergency vehicle detection using AI
- Dynamic traffic signal control
- Pollution monitoring using MQ-135 sensor
- IoT integration using ESP32

## 🛠 Tech Stack
- Python
- OpenCV
- ESP32 (IoT)
- Sensors (MQ-135)

## ▶️ How it Works
- Detects vehicles using camera input
- Identifies emergency vehicles
- Adjusts traffic signals in real-time
- Monitors air quality and sends alerts

## 🌍 Use Case
Improves traffic flow, reduces delays for emergency vehicles, and monitors environmental pollution.
